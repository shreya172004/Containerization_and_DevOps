Setup & Execution

01
Verify Docker Installation
Confirm Docker and Docker Compose are installed and running.
$ docker --version Docker version 29.2.1, build a5c7197 $ docker compose version Docker Compose version v5.0.2
02
Create IPvlan Network
Manually create the external IPvlan network before running Docker Compose.
$ docker network create -d ipvlan --subnet=172.22.208.0/24 --gateway=172.22.208.1 -o parent=eth0 mynetwork 3c13de8da92470216f345f5516e48ce390b1bdee209c09b5871211874e336062 $ docker network ls 3c13de8da924 mynetwork ipvlan local ✓
03
Build and Start Containers
Docker Compose builds both images and starts the services.
$ docker compose up --build [+] Building 14.2s (4/4) FINISHED ✔ Container postgres_db Healthy ✔ Container backend_api Created
04
Verify Static IP Addresses
Confirm both containers received their assigned IPvlan static IPs.
$ docker inspect backend_api | grep IPAddress "IPAddress": "172.22.208.11" $ docker inspect postgres_db | grep IPAddress "IPAddress": "172.22.208.10"
05
Test API Endpoints
Health check, insert record, and fetch records all working correctly.
# Health Check {"status":"healthy","service":"fastapi-backend"} # POST /api/records {"message":"Record inserted successfully","id":1} # GET /api/records [{"id":1,"data":"Hello from Shreya","created_at":"2026-03-19 18:25:55.631923"}]
06
Volume Persistence Test
Data survives complete container restart — confirming named volume works.
$ docker compose down ✔ Container backend_api Removed ✔ Container postgres_db Removed $ docker compose up -d ✔ Container postgres_db Healthy ✔ Container backend_api Created [{"id":1,"data":"Hello from Shreya","created_at":"2026-03-19 18:25:55.631923"}] ✅ Data persisted successfully! $ docker volume ls local containerized-webapp_pgdata ✓

IPvlan vs Macvlan

FeatureMacvlanIPvlan
MAC AddressUnique per containerShared with host
Network LoadHigher (more MACs)Lower
ScalabilityLimited by switchHighly scalable
Host IsolationHost isolatedHost can communicate
Best Use CaseSmall LAN / bare metalCloud / VMs ✓ Used here
